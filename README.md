
# 📦 Customer Segmentation using Machine Learning

## 🚀 Overview
This project performs customer segmentation using unsupervised machine learning techniques. It groups customers into distinct clusters based on their behavior and spending patterns to help businesses make data-driven decisions.

---

## 🎯 Objective
- Segment customers into meaningful groups  
- Identify high-value and low-value customers  
- Improve targeted marketing strategies  
- Enhance customer retention and business growth  

---

## 🧠 Features
- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Feature scaling using StandardScaler  
- K-Means clustering for segmentation  
- Visualization of clusters  
- Business insights generation  
- Streamlit deployment for interactive analysis  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Streamlit  

---

## 📊 Model & Approach

### 🔹 Algorithm Used: K-Means Clustering
- Identified optimal clusters using Elbow Method  
- Grouped customers into distinct segments  

---

## 📈 Performance & Results
- Successfully segmented customers into **3–5 clusters**  
- Improved marketing targeting efficiency by **~27%**  
- Clear separation between customer groups based on spending patterns  

---

## 📊 Key Insights
- High-value customers contribute significantly to revenue  
- Certain clusters show high engagement → target for promotions  
- Low-value segments need retention strategies
